**Preview images and icons directly from the Project View.**

## Features

Supports the following formats:

 - *.gif
 - *.png 
 - *.bmp
 - *.jpg
 - *.wbmp
 - *.svg
 - *.tiff
 - *.bigtiff
 - *.dcx
 - *.icns
 - *.ico
 - *.jbig2
 - *.pam
 - *.pbm
 - *.pcx
 - *.pgm
 - *.pnm
 - *.ppm
 - *.psd
 - *.rgbe
 - *.tga
 - *.tif
 - *.tiff
 - *.wbmp
 - *.xbm
 - *.xpm
